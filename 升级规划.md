## 📋 技术栈升级

| 模块 | 当前 | 升级后 |
|------|------|--------|
| 状态管理 | Zustand | Zustand + IndexedDB |
| 数据存储 | 内存 | electron-store + 加密 |
| 文件导入 | 手动输入 | xlsx + csv-parser |
| 并发控制 | 无 | p-queue |
| 日志系统 | console | winston |
