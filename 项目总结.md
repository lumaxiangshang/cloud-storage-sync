## 🚀 如何运行

### 开发模式
```bash
cd /root/.openclaw/workspace/cloud-storage-sync
pnpm install
pnpm exec playwright install chromium
pnpm dev
```

### 打包发布
```bash
# 构建
pnpm build

# 打包
pnpm dist
```

## 📝 技术要点

### Playwright 浏览器自动化
- 使用 `headless: false` 显示浏览器，方便处理验证码
- 每个网盘独立页面实例
- 支持用户手动介入登录

### Electron IPC 通信
- 主进程处理网盘操作
- 渲染进程负责 UI
- 异步任务状态更新

### TypeScript 类型安全
- 完整的类型定义
- 接口驱动开发
- 编译时错误检查
